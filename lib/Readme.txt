Install registry library
===========================
1. Open cmd/terminal.
2. Navigate to current folder.
3. Type: mvn install:install-file -Dpackaging=jar -Dversion=1.0.0 -DgroupId=com.ice.jni -DartifactId=registry -Dfile=registry-1.0.0.jar
4. Hit enter.

Please visit www.groupdocs.com for more information.
